# Live Talk Signaling Server

A high-performance Node.js signaling server for the **Live Talk** Android app.
Built for real-time video chat using **Socket.io** and **WebRTC**.

### 🚀 Public Deployment
Deployed to the internet via **Render.com**.
*   **Signaling Channel**: WebSockets & Polling
*   **Port**: Scalable to 3000
